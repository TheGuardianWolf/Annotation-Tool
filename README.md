# UoA Annotation Tool
forked from [angular2-webpack-starter](https://github.com/AngularClass/angular2-webpack-starter)

## Commands

```bash

# Install
npm install

# Build application
npm run build

# run electron application (no build)
npm start

# build and open electron
npm run electron

# Create electron package (win32)
npm run package

# Create electron package (linux)
npm run package-linux

```

## Runtime Dependencies

### Electron

Required to provide an interactive GUI for users.

### Q

A promise library to simplify async tasks and to convert node callback based async methods to promise async.

### Angular 2

Simplify app UI and app state binding, and provides dynamic routing to application pages.

### Bootstrap

Provides pre-formatted UI elements.

### CoreJS

Provides required polyfills for angular.

### jQuery

Simplifies DOM manipulation.

### rxjs

Reactive app state storage library.

### OpenSeaDragon

High level image viewer library for HTML5 canvas.

### Paper.js

High level vector manipulation library for HTML5 canvas.
